- ✅ 添加/取消收藏音乐

### 技术约束
- 用户ID固定为 0

- DELETE `/api/favorite/{musicId}` - 取消收藏

### 问题识别与修复

#### 🔴 P0 - 字段映射错误（已修复）

**问题描述**：收藏状态检查使用了错误的字段映射

**影响范围**：
- `favoriteIds` 集合构建错误，导致收藏状态判断失败
- 收藏按钮显示状态不正确
- 收藏列表删除逻辑错误

**修复文件**：
1. `src/stores/favorite.js:50` - 修复 favoriteIds 映射（id → musicId）
2. `src/stores/favorite.js:75` - 修复列表过滤逻辑（id → musicId）
3. `src/views/FavoriteList.vue:106` - 修复音乐ID传递（id → musicId）

**修复前**：
```javascript
favoriteIds.value = new Set(response.records.map((item) => item.id))
favoriteList.value = favoriteList.value.filter((item) => item.id !== musicId)
selectedMusicId.value = item.id
```

**修复后**：
```javascript
favoriteIds.value = new Set(response.records.map((item) => item.musicId))
favoriteList.value = favoriteList.value.filter((item) => item.musicId !== musicId)
selectedMusicId.value = item.musicId
```

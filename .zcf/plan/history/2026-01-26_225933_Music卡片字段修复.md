# Music卡片字段修复执行计划

## 任务上下文

**任务描述**：修复 MusicCard 组件中与后端 API 字段不匹配的问题

**创建时间**：2026-01-26 22:59:33

**问题分析**：
- 后端 API 返回字段：`title`（音乐名称）、`artistNames`（歌手名称）
- 前端组件使用字段：`name`、`artistName`
- 字段不匹配导致数据无法正确显示

**影响范围**：
- 文件：`src/components/MusicCard.vue`
- 影响行数：3处（第9、39、42行）

**采用方案**：方案1 - 最小化修改（直接修改字段引用）

---

## 执行步骤

### 步骤 1：修改图片 alt 属性（第9行）

**文件**：`src/components/MusicCard.vue`

**位置**：第9行

**修改前**：
```vue
:alt="music.name"
```

**修改后**：
```vue
:alt="music.title"
```

**说明**：将图片的 alt 属性从 `music.name` 改为 `music.title`，与后端 API 返回字段一致

**预期结果**：图片 alt 文本能正确显示音乐名称

---

### 步骤 2：修改音乐名称显示（第39行）

**文件**：`src/components/MusicCard.vue`

**位置**：第39行（h3 标签内）

**修改前**：
```vue
{{ music.name }}
```

**修改后**：
```vue
{{ music.title }}
```

**说明**：音乐名称标题从 `music.name` 改为 `music.title`，确保正确显示

**预期结果**：卡片中能正确显示音乐名称（例如："七里香"）

---

### 步骤 3：修改歌手名称显示（第42行）

**文件**：`src/components/MusicCard.vue`

**位置**：第42行（p 标签内）

**修改前**：
```vue
{{ music.artistName || '未知歌手' }}
```

**修改后**：
```vue
{{ music.artistNames || '未知歌手' }}
```

**说明**：
- 将 `music.artistName` 改为 `music.artistNames`（复数形式）
- 保持空值兜底逻辑：`|| '未知歌手'`
- 后端返回多个歌手时已用 "/" 分隔（例如："周杰伦/方文山"）

**预期结果**：卡片中能正确显示歌手名称，多个歌手时显示为 "歌手1/歌手2"
